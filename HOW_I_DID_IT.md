## What I did

- Built an AI agent using the LPI MCP server
- Queried multiple tools to gather structured knowledge
- Used Ollama as a local LLM for generating answers
- Added explainability by listing tool sources

## Problems I faced

- Ollama timeout due to large model and prompt size
- Fixed by using a smaller model and reducing prompt size

## What I learned

- Tool-based AI is more reliable than pure LLM responses
- Explainability improves trust in AI systems
